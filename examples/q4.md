## Background
Hybrid search is a technique that combines the strengths of sparse keyword retrieval (BM25) and dense vector search to improve recall and accuracy. BM25 is a sparse keyword matching approach that breaks down under vocabulary mismatch [13], while dense retrieval models can deteriorate in performance when encountering noisy user-generated text [12]. Hybrid search addresses these limitations by fusing the two approaches using reciprocal rank fusion and weighted score combination [21].

## Key Findings
Hybrid search combines sparse keyword retrieval (BM25) with dense vector search, fused via Reciprocal Rank Fusion (RRF) [1], to handle query types that break each approach in isolation [19].
